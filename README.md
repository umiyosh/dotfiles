## Clone from github and setup.sh and zsh

### Install

you must install zsh and vim before install these dotfiles.

~~~
  % git clone https://github.com/umiyosh/dotfiles.git
  % cd ./dotfiles/
  % ./setup.sh
  % zsh
~~~
